Operating System Support

The IEEE 1394 support libraries included with the application software you have installed have been certified on Windows 2000 and Windows XP Professional. We have not done anything purposefully to break Windows 98SE, Windows ME, or Windows NT 4.0 but no testing effort was applied to those operating systems. Windows 95 is not supported and there is no intent to support Windows 95 in a future release.

Installation

When you first plug in your new DCS 760, 760M, 720x cameras, Pro Back, or Pro Back Plus, the operating system will display the "New Hardware Found" dialog. To install the driver software for your new camera or back have the software CDROM you received with it in your CDROM drive and click on the appropriate link below to get instructions for installing a device driver. The folder where the drivers are located on the CDROM is "WDMDrivers\ProDriver". There is a folder for each operating system.

For Windows 98SE and ME

For Windows 2000 and XP
Scroll down the Window on the left side of the page and find "Control Panel". Under that is an icon named "Add Remove Hardware". Left click the + sign to expand the menu choices. Double click the "Installing Devices" option under the "Install or Uninstall Device" option. Once this page opens, select the option at the bottom titled "Install a device".

If you have a legacy camera (DCS 315, 330, 520, 560, 620, 620x, 660, or 660M) and plan on simultaneously tethering it to your PC with your new camera or back, it is required that you upgrade the driver you received previously. To complete the upgrade, have the software CDROM you received with your new camera or back in your CDROM drive. Have the legacy camera you wish to upgrade the driver for attached. Click on the appropriate link below to get the instructions for updating a driver. The folder where the updated drivers are located on the CDROM is WDMDrivers\DCSDriver. There is a folder for each of the operating systems.

Limitations

These new versions of the IEEE 1394 drivers do not support "Hot Pluggability". What does this mean? When the application is active, connecting new cameras or disconnecting currently attached cameras is not supported. It is highly recommended that you not attempt this, as it will potentially hang the system forcing a reboot. If you need to connect other cameras, or disconnect any that you have connected, you should exit the application first. After you have completed your connection/disconnection action then you can reactivate the application. This will be corrected in a future release.

At this time Direct Memory Access (DMA) is not supported on Windows 98SE, ME, 2000, and XP. This means that the transfer time of images from the camera's or back's memory or PCMCIA card to a location on your hard drive is not optimized. This will be corrected in a future release. Typical speeds we are seeing is 3.5 to 4.0 MB/sec on a 500MHz Pentium II machine with 128MB of memory.
