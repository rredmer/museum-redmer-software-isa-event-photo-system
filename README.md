## Introduction

The Event Photo System for ISA by Redmer Software Company is designed to manage and process event photos efficiently. This system includes functionalities for capturing, storing, and retrieving event photos, along with various utilities to enhance and manage the images.

## Features

- Capture and store event photos
- Retrieve and display stored photos
- Manage photo metadata
- Support for multiple image formats
- Integration with Kodak DCS cameras
- User-friendly interface for managing photos
- Database support for storing photo information

## Installation

To install the Event Photo System, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/museum-redmer-software-isa-event-photo-system.git
    ```

2. Navigate to the project directory:
    ```sh
    cd museum-redmer-software-isa-event-photo-system
    ```

3. Install the necessary dependencies:
    ```sh
    npm install
    ```

4. Build the project:
    ```sh
    npm run build
    ```

## Usage

To use the Event Photo System, follow these steps:

1. Start the application:
    ```sh
    npm start
    ```

2. Open your web browser and navigate to `http://localhost:3000`.

3. Use the interface to capture, store, and manage event photos.
